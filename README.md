## sbt-kalix-improving Template 
In src/main/g8 there is a giter8 template for creating new
multiple Kalix service projects that use the sbt plugin with `sbt new`.

Usage:
```shell
> sbt new improving-ottawa/sbt-kalix-improving 
```
